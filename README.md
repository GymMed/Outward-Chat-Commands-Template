Manager](https://github.com/GymMed/Outward-Chat-Commands-Manager).
After that, you can use the commands provided by this mod directly in chat.

## How to set up

To manually set up, do the following

1. Create the directory: `Outward\BepInEx\plugins\OutwardChatCommandsTemplate\`.
2. Extract the archive into any directory(recommend empty).
3. Move the contents of the plugins\ directory from the archive into the `BepInEx\plugins\OutwardChatCommandsTemplate\` directory you created.
4. It should look like `Outward\BepInEx\plugins\OutwardChatCommandsTemplate\OutwardChatCommandsTemplate.dll`
   Launch the game.
